# 🚀 QR Code Generator

A modern and responsive QR Code Generator built with React and Vite.

## ✨ Features

- 🔗 Generate QR codes from URLs or plain text
- 📥 Download QR codes as PNG
- 🧹 Clear input with one click
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and intuitive user interface
- ✅ Input validation for a better user experience

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
